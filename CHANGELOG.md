# Changelog

All notable changes to this project will be documented in this file.

## v1.0.0 - 2026-04-06

### Added
- Next.js App Router portfolio covering landing, project, and docs surfaces.
- Section-based landing page with hero, project, about, and contact sections.
- Arc Timer case study route with product overview, screenshots, feature breakdown, and implementation notes.
- Project-scoped docs workspace for Arc Timer reference material.
- Registry-driven docs navigation supporting project docs and standalone docs routes.
- Demo docs fixtures gated behind `ENABLE_DOCS_DEMOS`.
- Theme persistence through cookies and `localStorage`.
- Centralized site metadata for page titles, Open Graph, and Twitter cards.
- Generated `opengraph-image`, `twitter-image`, `robots.txt`, and `sitemap.xml` routes.
- Shared theme tokens, motion configuration, and CSS variable-driven styling.
